# Dankal Java 组招聘岗位


## 一、 初级 Java开发工程师
```
岗位要求:
1.良好的Java语言基础;
2.掌握J2EE相关知识，熟悉常用JavaWeb框架
3.熟悉Spring, Spring MVC, MyBatis等开发技术；
4.熟悉的常用网络协议（HTTP、TCP等）;
5.掌握常用数据库理论和使用，如MySQL、Oracle等其一;
6.熟悉nosql，如mongodb，redis等其一；
7.有个人博客者优先
```

## 二、 中级 Java开发工程师
```
岗位要求:
1.具备2年以上Java项目开发经验；
2.扎实的Java语言基础，了解面向对象和设计模式，能深入了解多线程编程原理；
3.熟悉SSM/SpringBoot框架开发；
4.精通MySQL数据库应用，对高并发，数据库集群有具体实践经验；
5.熟悉nosql，如mongodb，redis等；
6.熟悉常用技能Maven、Git、IDEA等；
6.熟悉网络通讯协议TCP/IP，HTTP等；
7.熟悉Restful接口设计，开发；
8.优先查看博客，github等资料；
9.熟悉Linux相关常用操作命令；
10.学习能力强，拥有优秀的逻辑思维能力，自我管理能力强，有良好的时间意识有较好的沟通交流能力，工作认真，细心，有条理；积极性高，求知欲强；具有较强的沟通能力以及团队合作精神；
```


## 三、高级 Java开发工程师

```
岗位要求：
1.本科或以上学历，计算机软件或相关专业，5年以上J2EE项目开发经验；
2.熟悉J2EE、J2SE规范体系及其设计模式、熟悉分布式应用系统的分层模式，熟悉SSM/SpringBoot框架开发；
3.熟悉Linux系统操作，知道在Linux环境下配置相关开发、测试环境；
4.熟悉Mysql、oracle等关系数据库以及非关系型数据库(Redis、MongoDB、HBase等) 之一种或者多种的应用程序开发，具有较强的设计能力，能够独立完成应用模块的设计和开发；
5.有大规模并发系统的开发经验，精通网络和服务器优化、缓存、分布式存储技术，对并发处理、安全处理、性能调优、测试分析有相关的实践经验，有千万级数据读取运行的优化经验优先考虑，有高性能的分布式系统设计及应用经验更佳拥；
6.具有主流内存及文件搜索引擎（Solr、Elasticsearch等）、分布式服务框架（Dubbo、Spring Cloud等）实施经验，并了解其原理；
7.有项目管理和带领团队经验，能熟练使用常见的项目和代码管理工具，如Git，Jenkins，Docker等；
8.优先查看博客，github等资料或开源项目；
9.拥有优秀的逻辑思维能力，自我管理能力强，有良好的时间意识有较好的沟通交流能力，工作认真，细心，有条理；积极性高，求知欲强；具有较强的沟通能力以及团队合作精神；
```
